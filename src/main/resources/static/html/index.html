Bem-vindo ao Sistema de Controle de Estoque

Sistema desenvolvido para facilitar o gerenciamento de materiais utilizados na construção civil.

Produtos

Cadastre novos produtos, altere informações e consulte os itens disponíveis.

Fornecedores

Gerencie todos os fornecedores responsáveis pelo abastecimento do estoque.

Movimentações

Registre entradas e saídas de produtos para manter o estoque atualizado.

Estoque

Consulte a quantidade disponível dos produtos cadastrados.



Objetivo do Sistema

O sistema tem como objetivo controlar os materiais da empresa, permitindo o cadastro de produtos e fornecedores, o registro das movimentações de entrada e saída e a consulta do estoque disponível.
